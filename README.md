# Library Management System

A C++ library management system implementing 
object-oriented design with inheritance and polymorphism.

## Features
- Media types: Book, DVD, Magazine
- Base class Medium with virtual functions
- Dynamic memory management with pointers
- Date handling with Datum class

## Technologies
- C++, OOP, Inheritance, Polymorphism
- RWTH Aachen University — Informatik Praktikum
